Elementos de un sistema de informaciòn computacional

1.1Equipo Computacional:

Hardware necesario para que el sistema de información funcione, integrado por computadoras y equipo periférico conectado a ellas.

1.2 Recurso Humano:

Son las personas que interactúan con el sistema de información suministran datos o utilizan el sistema para generar datos.

1.3 Fuente:

Son los datos de información que se introducen en el sistema, es decir, son las entradas requeridas para que el sistema informático funcione.

1.4 Programas:

Software ejecutado por la computadora, el cual procesa la imformación de entrada para generar los resultados esperados.

1.5 Telecomunicaciones: Abarca el software y hardware que permite la transmisión en forma electronica de texto e imagenes, entre otros.

1.6 Politicas y reglas de operación:

Aplicada alos procedimientos y mecanismos para hacer que una aplicación de computadora trabaje de acuerdo con estas o regir la parte funcional del proceso administrativo de la empresa.
